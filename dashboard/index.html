Why This Matters at NYT
⚠️ Before: What Happened
Games team renames game_name → game_title in mobile SDK on a Friday. Monday 6 AM, dbt staging runs, JOIN on renamed column silently returns NULL. Games WAU drops from 2,953 to 0. CEO asks why in Monday review.
✅ After: What Happens Now
Schema drift detector runs as Task 0 every day before any dbt model. Detects game_name is MISSING_COLUMN at 6:01 AM. Airflow blocks staging. Slack pings the Games team. Fix is deployed by 8 AM. Monday review uses correct data.
📋 Registry as Source of Truth
Schema changes now require a PR to the schema registry — reviewed by Data Platform before the SDK ships. Breaking columns are version-controlled in Git alongside the dbt models they feed.

The Problem: 5 Teams, 5 Different Numbers on the Same Monday Report
🎮 Games Team
COUNT(game_start events)
Inflated — counts retries, rage-clicks
📰 News Team
COUNT(article_view events)
Inflated — includes bots, crawlers
🏆 Athletic Team
COUNT(liveblog_view) — no time gate
Arbitrary — 1 second counts as active
💰 Finance
active subscription day — no engagement
Misleading — churned sub counts as active
📖 Editorial
any pageview incl. homepage bounce
Wildly inflated — 3× other counts
Official Definition — One SQL File, All Teams Sign Off, Version in Git
📰 News
article_complete
Reading = completing. Landing on an article doesn't count. Eliminates bots and click-throughs from social.
🎮 Games
game_complete
Intentional engagement only. Rage-quits and partial attempts excluded. Streak milestones also qualify.
🍳 Cooking
cook_mode_start
High intent signal. Browsing recipes ≠ active. Entering cook mode = user committed to the experience.
🏆 Athletic
article_view + time > 60s
Quality gate for short-form liveblogs. Aligns with industry standard bot-exclusion threshold at 60 seconds.
# Key engineering decision:
A subscriber active in Games AND News on the same day = 1 DAU, not 2.
cross_product_total CTE deduplicates across all products before aggregation.
# Version-pinned:
Every row carries _metric_version = '2024.Q1' so downstream consumers can detect definition changes via JOIN.

AIRFLOW
Why Yesterday's Numbers Are Wrong Without This
A subscriber plays Wordle Tuesday 11 PM on airplane mode. Phone reconnects Wednesday 7 AM.

Without reprocessing: Tuesday's DAU is permanently understated. That subscriber triggers a false churn alert. Retention sends an unnecessary win-back email.

With reprocessing: Wednesday's 6:05 AM Airflow run detects their late event and reruns Tuesday's partition. DAU is corrected before the retention team's 9 AM standup.
